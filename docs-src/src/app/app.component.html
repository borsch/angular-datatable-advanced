Angular Datatable Advanced

About

Simple datatable with filtering, sorting and server-side data loading. Supports column-based configuration. Has few different filter types:
  • Text
  • Filtering with multi-select. support filter loading via Observable
  • Date & Date-Time picker. As picker Flatpickr is used with angularx-flatpickr as wrapper

Simple table example

Table with example of:
  • Plain text column rendered by field name(first column)
  • Column rendered with nested field(second column)
  • Column rendered with custom cellRender function(third column)
Code example 
minExampleColumns: Column[] = [
  {
    columnKey: 'plainTextFilter',
    columnName: 'Plain Text filter',
    sortable: true,
    filterable: true
  },
  {
    columnKey: 'render.deep.field',
    columnName: 'Render deep field',
    sortable: true,
    filterable: true
  },
  {
    columnKey: 'cellRenderExample',
    columnName: 'Custom cellRender Example',

  }

Set filter example

Table with example of:
  • Set filter with inline array of options(first column)
  • Set filter with options loaded with Observable(second column)
Code example 
import {FilterIn} from 'angular-datatable-advanced/lib/model/filter-in';

columnSetSelect: Column[] = [
  {
    columnKey: 'setFilter',
    columnName: 'Set Filter',
    filterable: true,
    filterIn: [{
      value: 'value_1',
      label: 'Readable label: value 1'
    }, {
      value: 'value_2',
      label: 'Readable label: value 2'
    }]
  }, {
    columnKey: 'setFilterObservable',
    columnName: 'Set filter with options from observable',
    filterable: true,
    filterInPromise: this.setFilterObservable()
  {
];

private setFilterObservable(): Observable<Array<FilterIn>> {
  // can be replaced with any http request
  return of([
    {
      value: 'observable_value_1',
      label: 'Observable value 1 label'
    }, {
      value: 'observable_value_2',
      label: 'Observable value 2 label'
    }
  ]);
}

Installation

npm install --save angular-datatable-advanced

You'll also need to install dependencies

Angular related dependencies: 

npm install --save @angular/cdk @angular/forms @angular/material

Flatpickr dependencies: 

npm install --save flatpickr angularx-flatpickr

Setup

Add Material CSS & Flatpickr CSS to angular.json to styles block. You can select another available Material There
}
  "styles": [
    // .. other css ...
    "./node_modules/flatpickr/dist/flatpickr.css",
    "./node_modules/@angular/material/prebuilt-themes/indigo-pink.css",
    "src/styles.css"
  ]
}
Add imports to app.module.ts
@NgModule({
  imports: [
    // .. other imports ..
    FormsModule,
    FlatpickrModule.forRoot(),
    MatIconModule,
    MatTableModule,
    MatPaginatorModule,
    MatSortModule,
    MatCardModule,
    MatMenuModule,
    MatProgressSpinnerModule,
    AngularDatatableAdvancedModule
  ],
})
export class AppModule { }
Create array of columns
import {Column} from 'angular-datatable-advanced';

columns: Column[] = [
  {
    columnKey: 'column1Key',
    columnName: 'Column 1 header'
  },
  {
    columnKey: 'column2Key',
    columnName: 'Column 2 header'
  }
];
Add rows loader
import {RowsLoader, LoadResult, RequestParams} from 'angular-datatable-advanced';
import {Observable, of} from 'rxjs';

rowsLoader: RowsLoader = (filters: RequestParams): Observable<LoadResult> => {
  // can be replaced with HttpClient call to backend server
  return of({
    totalItems: 1,
    rows: [
      {
        column1Key: 'Column 1 value',
        column2Key: 'Column 2 value'
      }
    ]
  });
}
Render table
<ada-table
  [pageSize]="10" // default page side
  [pageSizeOptions]="[10, 20]"  // page size options for pagination component
  [columns]="columns"
  [rowsLoader]="rowsLoader"
>
</ada-table>
